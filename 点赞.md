# 点赞模块设计
## 1. 视频点赞 、douyin/favourate/action
**输入输出参数**：
```go
// 传入参数
type douyin_favoriate_action struct {
    user_id     int64   // 用户 ID
    token       string  // 用户鉴权 token
    vedio_id    int64   // 视频 ID
    action_type int32   // 1 点赞 2 取消
}

// 返回参数
type douyin_favoriate_action_response struct {
	status_code int32   // 状态码
	status_msg  string  // 状态信息
}

// 被点赞视频的作者的用户详细信息
type User struct {
	Id             int64        // user ID
	Name           string       // username
	FollowCount    int64        // user 的关注数
	FollowerCount  int64        // user 的粉丝数
	IsFollow       bool         // cur 是否关注 user
	TotalFavorited int64        // user 的视频被点赞的总量
	FavoriteCount  int64        // user 点赞过的视频总量
}

// 点赞数据结构
type Like struct {
	Id      int64       // 点赞 ID
	UserId  int64       // cur ID
	VideoId int64       // 视频 ID
	Cancel  int8        // 0 点赞，1 取消
}
```
**详细流程**
```go
// Gin 路由组监听点赞视频事件
apiRouter.POST("/favorite/action/", jwt.Auth(), controller.FavoriteAction)
// 解析 cur_id，video_id，
```
